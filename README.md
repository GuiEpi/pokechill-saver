![pokechill-saver](assets/banner.png)

A browser extension that auto-saves your [pokechill](https://play-pokechill.github.io/) game data to Dropbox.

## Features

- **Save to Dropbox** - Back up your game data with one click
- **Load from Dropbox** - Restore your save on any browser
- **Auto-save** - Automatically saves when you leave the game page

## Usage

1. Open [pokechill](https://play-pokechill.github.io/)
2. Click the extension icon
3. Connect to Dropbox
4. Use **Save** / **Load** buttons, or just play — it auto-saves when you leave the page
